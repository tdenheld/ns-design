Guidelines

  • Never use more then one primairy button on a single page. Except a long page where the same call to action destination is repeated at the bottom of the page.
  • Don’t use classic disable buttons at all by default. If a buttons gets disabled it keeps its color and gets a spinner (without text) – except for the tertairy button because it’s only used next to another button. When multiple buttons are displayed, only the clicked buttons gets a spinner.
  • When leading to a buy funnel – not a subscription overview for example – a cart icon must be added. Colour priority: yellow, primairy, secundairy depending on design.
  • Only use arrows in the funnel itself and in combination with brand button (yellow).
  • Default no icons. Icons are optional. If use an icon, only use one and before text. Except arrows: they must be placed after the text. Never use an icon and arrow at the same time.
  • Don’t use subtle buttons on an image background – you can use them on yellow or light grey.
  • Use text / icon colour as displayed in previews above.
  • Never use a secondary button for submitting a form
  • Brand (yellow) button is used for: hero/banner, entering funnel and funnel itself.
  • Floating buttons are used for maps.
